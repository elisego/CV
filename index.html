Projets

Pendant notre formation nous avont réalisés 3 projets et 2 hackathons:

Projet 1: Nous avons crée le site d'un restaurant voisin de l'école, en utilisant: HTML, CSS - 1 semaine Le dépot du site est disponible sur Github.

Projet 2: Nous avons crée un site style Stackoverflow. Le site permets de créer un profil, de poser et répondre à des questions. , en utilisant: HTML, CSS, JavaScript, MySQL - 5 semaines Le dépot du site est disponible sur Github.

Projet client: Nous avons crée le site Wesource. C'est une plateforme pour mettre en relation les fabricants de tissus et les marques de vêtements. Le site permet de rechercher des produits, de consulter les fiches produits et de consulter les pages "Company". De plus nous avons créés une partie administration qui permets de créer et modifier les produits, les companies et les profils. Nous avons utilisés: HTML, CSS, JavaScript, Materialize, Hibernate, MySQL, Spring Security Le dépot du site est disponible sur Github.

Hackathon n°1: StarLove! Création d'un site sur 2 jours. Le but était de créer un site en un temps très limité en utilisant la base de donnée mise à disposition. Nous avons créés un site pour trouver l'amour dans le monde de la saga Star Wars en utilisant: HTML, CSS, JDBC, MySQL. Le dépot est sur Github.

Hackathon n°2: Pilulib! Création d'un site en moins de 3 jours. Le but était de créer un site sur le thème de la santé et du remote. Nous avons crées un site permetant aux personnes agées de remplir leurs piluliers en suivant les indications du site. Nous avons utilisés: HTML, CSS, hibernate, JavaScript, MySQL, Materialize. Le dépot est sur Github.
